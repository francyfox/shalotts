# Shalotts [WIP]

`Inspered by BatiJS`

With this boilerplate generator you can create modern scaffolding tool for your JS framework/library on monorepo.

Works with

- Bun Workspace
- Turborepo (soon)
- NX (soon)

How it Works?

- Create git templates with Tera (jinja2) and Toml configs
- Set special tag for git `$sha_template`
- Test with coderabbitai
- Release

Benefits:

- Create your own preset and share it with special DNA code. For monorepo creation paste your DNA code to CLI
- Simple append to your monorepo apps and packages from market store. Your backend generator easily transforms to full-stack application.
- You don't need to worry about creation npx commands for code generation
- Self-hosted
